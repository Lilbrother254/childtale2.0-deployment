# ChildTale Art Samples

This folder contains sample coloring pages to showcase ChildTale's art quality and variety.

## File Naming Convention

Please name your files descriptively:
- `adventure-[name].jpg` - Adventure/action themes
- `animals-[name].jpg` - Animal characters
- `fantasy-[name].jpg` - Fantasy/magical themes
- `everyday-[name].jpg` - Everyday life scenarios
- `sports-[name].jpg` - Sports/activities

## Recommended Format

- **Format**: JPG or PNG
- **Resolution**: 1200px width minimum (for quality display)
- **Orientation**: Portrait preferred (matches book format)

## Examples

```
samples/

  adventure-space.jpg
  animals-safari.jpg
  animals-ocean.jpg
```
