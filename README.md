# ChildTale - Personalized Coloring Book Creator üé®

ChildTale is a magical application that uses AI to generate personalized coloring books for children. By combining the power of **Google Gemini**, **Supabase**, and vivid creativity, parents can create unique stories where their child is the main character.

## ‚ú® Features

- **Personalized Stories**: Create stories featuring your child's name, age, gender, and description.
- **Two Magic Modes**:
    - **Standard Mode (Free)**: Quick, 5-page story generation.
    - **Deep Dive Mode (Premium)**: In-depth, 25-page adventure with "Meaningful Screentime" features.
- **AI-Powered Illustrations**: High-quality, colorable line art generated by Gemini.
- **Magic Studio**: A dedicated space to read, color, and manage generated books.
- **Library**: Persistent storage for all your creations.
- **Hardcover Printing**: Integration with Lulu for printing real physical books (in progress).
- **Secure Authentication**: Magic link and password login via Supabase.

## üöÄ Tech Stack

- **Frontend**: React, TypeScript, Vite, Tailwind CSS
- **Backend/Database**: Supabase (PostgreSQL, Auth, Storage, Edge Functions)
- **AI**: Google Gemini Pro & Flash
- **Payments**: PayPal Integration
- **Printing**: Lulu Direct API

1.  **Clone the repository**:
    ```bash
    git clone https://github.com/Lilbrother254/childtale2.0-deployment.git
    cd childtale2.0-deployment
    ```

2.  **Install dependencies**:
    ```bash
    npm install
    ```

3.  **Configure Environment Variables**:
    Create a `.env.local` file in the root directory and add your credentials:
    ```env
    VITE_SUPABASE_URL=your_supabase_project_url
    VITE_SUPABASE_ANON_KEY=your_supabase_anon_key
    ```

4.  **Run the development server**:
    ```bash
    npm run dev
    ```

## üåç Deployment

This project is configured for deployment on **Vercel**.

1.  Connect your GitHub repository to Vercel.
2.  Add the same environment variables (`VITE_SUPABASE_URL`, `VITE_SUPABASE_ANON_KEY`) in the Vercel dashboard.
3.  Deploy!

## üîê Security

- **Row Level Security (RLS)**: Enabled on all Supabase tables to ensure users only access their own data.
- **Edge Functions**: Sensitive operations (payment processing, book management) are handled securely on the server side.
